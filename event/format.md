# Discussion Night — Event Format

## The event

A public gathering to ask one question together: **"What does it mean to be human in the AI era?"**

Not a lecture. Not a debate. A conversation — for everyone navigating AI in 2026.

**Duration**: 2–2.5 hours  
**Attendees**: 15–200  
**Setup**: Minimal — tables for small groups, name tags, a screen for the presentation

---

## Agenda

| Time | Block | Notes |
|------|-------|-------|
| 0:00 – 0:40 | **Networking** | Doors open, name tags at the entrance, music playing. Let conversations warm up naturally. |
| 0:40 – 1:00 | **Opening keynote** | HQ presentation — who we are, why this week, what we're asking. Chapter lead delivers this. |
| 1:00 – 1:45 | **Breakout discussions** | Small groups (5–8 people) discuss 2–3 questions. Rotate optional. |
| 1:45 – 2:00 | **Reconvene + close** | Groups share one insight. Chapter lead closes with the pledge CTA. |

---

## Setup requirements

**Must have:**
- A venue (room, café, office, coworking space — anything works)
- A screen or TV for the presentation
- Name tags (see below)
- WiFi if using the [conversation-prompter app](../app/README.md)

**Nice to have:**
- Food and drinks (covered by local sponsor — see [`sponsorship/pitch.md`](../sponsorship/pitch.md))
- Printed question cards for each table (optional — pull from [`content/questions.md`](../content/questions.md))

---

## Name tags

Color-coded name tags are a simple way to spark cross-pollination in the room.

| Color | Role |
|-------|------|
| 🔵 Blue | Founders / Builders |
| 🟣 Purple | Investors |
| 🟢 Green | Researchers / Academics |
| 🟡 Yellow | Operators / Practitioners |
| 🔴 Red | Policy / Government |
| ⚪ White | Everyone else / Curious public |

Avery labels, sticky notes, or colored dot stickers all work. Total setup time: ~10 minutes.
