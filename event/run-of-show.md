# Run of Show — Discussion Night

Copy this and fill in your local details before the event.

---

## Pre-event (day before)

- [ ] Confirm venue is set and accessible
- [ ] Confirm sponsor logistics (food/drinks delivery, setup time)
- [ ] Print or prep name tags (color-coded by role)
- [ ] Test the presentation on the venue screen
- [ ] If using the app — deploy or have the URL ready: [conversation-prompter](../app/README.md)
- [ ] Share event reminder with your RSVP list

---

## Day of — setup (60 min before doors)

- [ ] Arrive at venue
- [ ] Set up name tag station at the entrance
- [ ] Arrange seating in small clusters (5–8 per table/group)
- [ ] Set up screen, connect laptop, open presentation
- [ ] If using app — open on a secondary screen or have URL on table cards
- [ ] Food/drinks set up and ready
- [ ] Music on

---

## Run of show

**[TIME] Doors open — Networking**
- Welcome guests at the door
- Direct them to name tags
- Mingle, no formal program yet

**[TIME + 40 min] Opening keynote**
- Gather everyone's attention (no mic needed under ~50 people)
- Deliver the HQ presentation (~15–20 min)
- End with: "We're going to break into small groups now. The question we're asking tonight is: what does it mean to be human in the AI era?"

**[TIME + 60 min] Breakout discussions**
- Groups of 5–8, ideally at tables
- Post questions on screen (app or slides) or place printed cards on tables
- Circulate and check in with groups
- Give a 5-minute warning before reconvening

**[TIME + 105 min] Reconvene**
- Ask each group to share one insight or surprising moment
- Brief, no pressure — just listening

**[TIME + 115 min] Close**
- Thank sponsors, venue, attendees
- Share the pledge: "If tonight resonated with you — add your name at hiai.com. Tell us what you believe about the future of AI."
- Announce any upcoming local events

**[TIME + 120 min] Open networking / close**
