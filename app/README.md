# Conversation Prompter App

An optional tool for displaying discussion questions during the breakout portion of your event.

**Repo:** [the-ai-collective/conversation-prompter](https://github.com/the-ai-collective/conversation-prompter)

---

## What it does

Displays AI-generated conversation questions on a screen for your breakout groups. Chapter leads or attendees can cycle through questions, rate them, and the system learns which questions land best.

---

## Using it at your event

**Option A — Self-hosted**: Deploy your own instance before the event (see the repo README for setup instructions). Requires a PostgreSQL database and an Anthropic API key.

**Option B — Printed questions**: If you don't want to set up the app, pull questions from [`content/questions.md`](../content/questions.md) and print them as table cards, or just display them in the presentation.

The app is a recommendation, not a requirement. The event works fine without it.
